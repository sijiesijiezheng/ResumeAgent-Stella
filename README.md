# Resume Agent v3

> 只需一点信息，生成你的智能简历。

## 特性
- 四级激活码体系：FREE / CAMPUS / PRO / DEV
- 一键导出 PDF、Word 或复制内容
- 无需配置环境变量，直接部署 Vercel

## 启动方式
1. 上传仓库至 GitHub。
2. 在 Vercel 选择 “Import Git Repository”。
3. 部署完成后访问你的域名即可使用。

## API 配置
```
const API_URL = 'https://resume-api.sijiesijiezheng.workers.dev';
const MODEL = 'gpt-4o-mini';
```
